Flexbox and Grid are both CSS layout modules used to create dynamic webpages, but they have distinct differences. Flexbox is a single-dimensional layout module designed to arrange elements into rows or columns along a specific axis. On the other hand, Grid is a two-dimensional layout module, allowing for elements to be placed into columns and rows simultaneously.

Unlike Flexbox, Grid includes features such as gaps and grid lines, which allow for more precise positioning of elements on the page. Ultimately, the decision to use Flexbox or Grid depends on the desired design of the webpage and how it needs to be laid out.

The purpose of media query is to provide a way for web developers to tailor their websites to specific devices and viewport sizes. By using this feature, developers can create different stylesheets and contents depending on the size of the device's screen or window and provide a better experience for their users.

Media queries also allow web developers to create responsive design which is optimized for different types of devices ranging from desktop computers to smartphones and tablets.

The CSS box-model is a system of styling HTML elements. It consists of four main components: margin, border, padding, and content. Margin defines the distance between an element and other elements around it; borders create a line around the element; padding adds space between the border and the content itself; and content is the actual material inside the element.

Together, these four areas work together to create a design that can be easily adjusted and manipulated with different properties in order to achieve various effects.

Semantic tags are HTML elements that provide meaning to the web page. They provide structure and help search engines better understand the content on the page. Semantic tags can also be used to define headings, paragraphs, lists, etc. which make it easier for users to navigate through a website or web application.
